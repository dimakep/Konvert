# Konvert  
Программа создана для печати отправителя и получателя на конвертах.  
Пока создана только для одного отпавителя, в будущем исправлю для добавления отправителя при установке.  
Сейчас на выбор три конверта:  
110х220 мм - с полями Область, Район, Город и тд.;  
110х220 мм - с полями Кому, Куда;  
160х230 мм - с полями Кому, Куда.  
dbo.Recipient.sql - скрипт создания базы данных
